# Cisco Packet Tracer Labs

This repository contains hands-on networking projects built using Cisco Packet Tracer, focused on real-world network design, configuration, and troubleshooting.

---

## 🧠 About

This is a collection of labs where I design, configure, and troubleshoot different network setups. Each project builds practical skills and reflects real-world networking scenarios.

---

## 📂 What’s Covered

* VLAN configuration and segmentation
* Inter-VLAN routing (Router-on-a-Stick)
* DHCP and IP address management
* Wireless networking and access point setup
* CCTV network design using static IPs
* Network security using ACLs
* Troubleshooting connectivity issues

---

## 🏗️ Approach

Each project follows a simple process:

1. Design the network
2. Configure devices (routers, switches, APs)
3. Assign IP addressing
4. Test connectivity
5. Troubleshoot issues

---

## 🎯 Goal

To build strong practical networking skills, gain hands-on experience, and develop a professional portfolio for network engineering roles.
